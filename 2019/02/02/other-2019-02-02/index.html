关于学习

发表于 2019-02-02 | 分类于 other | 阅读次数: |
字数统计: 973 | 阅读时长 ≈ 3

引言


    趁着放假了终于有时间整理一下过去的2018，总的来说过的很忙碌、很充实。从结果来看还算是令人满意的，回想起来还有一些方面还能够做的更好或者需要加强，2018年的计划也只完成了一部分-_-||。

温故知新很重要


    对于学习，自认为是一个爱好学习的人，但是就这一点来讲还远远不够。如今这是信息爆炸时代，知识是学不完的，现如今想来，在学习的同时温故知新很重要，将学习的东西转换成自己的知识这才是长久之道。这点感悟颇深，刚开始工作时主要是是用C/C++开发，对于内存管理啊、C++友元函数、友元类的用法啊可以说是信手拈来，上次同事问我时，我发现我学的这些知识已经记忆模糊了，我反正跟她是形同陌路，令人汗颜。也好久没有写博客了，准备写一篇，发现Markdown好多语法都忘了、忘了……，类似于这样的情景时常出现，我是要感叹我老了吗？o(╥﹏╥)o，所以后续的策略只有对于一些重要的东西只有多练习、多复习。想起来看动漫《一人之下》里面的宝儿姐的一个“癖好”——喜欢埋人。张楚岚问她，你为何这么快？ 她说：”唯手熟尔！”

环境很重要


    孟母三迁的故事想必我们中学时代写作文都已经写烂了。其实发现工作中环境确实也很重要，跟优秀的人交流、学习、相处，以他/她们为榜样，想必过一段时间你自然会变得更优秀。大家都在“打农药”、刷抖音如何才能快速融入他们？当然是加入他们。（手动滑稽），如果想要学习，找一个安静的地方，断其wifi（试验过一天不带手机，回来后发现真没人发消息或者打电话给你。），持续一段时间相信必然卓有成效。

抵住诱惑


    现如今诱惑真的太多了。在抖音里有漂亮的你见不了的真人小姐姐；有看了学不会的技能，有过不了也玩不了的别人的那令人羡慕的生活（我真的不是为它打广告！真的！）； 在游戏里你可以所向披靡，掌控雷电。在网络小说里面你可以掌控三界、叱咤风云、美女财富如云。以前真不知道自己想要什么，做什么，现在已经越来越清晰了。

社会不是理想国，活下去才是最重要的


    当然啦，在学习的过程中，玩耍、休闲、娱乐、游戏这些是必要的也是必须的。如果脱离了这些你可能已经跟社会脱节。但是其中的度需要自己把握，这个很重要,并且前提是你的有个目标。我是很推崇“中庸之道”，所谓：”中庸之为德也，其至矣乎!民鲜久矣。”,朝着一个目标奋斗，虽然很难，但是能做到就很了不起了。

写在最后


    学习真的是一辈子的事，以前是学习书本上的，现在不仅学习书上的，也需要向其他人学习，三人行必有我师。后续会整理这些年学的专业知识、看的书、个人感悟将其记录下来，算是温故而知新吧。
